## 技术栈

- **视频处理**：OpenCV
- **目标检测**：YOLOv8 (ultralytics)
- **目标跟踪**：卡尔曼滤波 + IOU匹配
- **轨迹平滑**：卡尔曼滤波
- **轨迹分析**：NumPy向量计算
- **可视化**：OpenCV + Matplotlib
- **数据输出**：Pandas
